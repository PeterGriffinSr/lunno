Module Lunno_frontend.Debug

val dump_tokens : Stdlib.Lexing.lexbuf -> unit

dump_tokens lexbuf reads and prints all tokens from the given lexing buffer until the end of file.

This function repeatedly calls the lexer to obtain tokens and prints their string representations to standard output. It continues until it encounters the EndOfFile token.

  • parameter lexbuf

    The lexing buffer containing the input source code.

  • returns

    Unit.

val dump_program : Ast.program -> unit

dump_program prog prints the structure of the entire AST program prog.

This function iterates over all top-level expressions in the program and invokes dump_expr to print each one.

  • parameter prog

    The AST program to dump.

  • returns

    Unit.

val string_of_ty : Ast.ty -> string

string_of_ty ty returns a human-readable string representation of the type ty.

  • parameter ty

    The AST type to convert.

  • returns

    A string representing the given type.
