Module Lunno_driver.Flags

type t = {
  1. dump_program : bool;
  2. files : string list;
}

The parsed CLI flags and positional arguments.

  • dump_program — whether --dump-program was passed
  • files — the list of input source file paths
val parse : string array -> t

parse argv parses argv (i.e. Sys.argv) into a t.

Handles --help/-help and --version/-v directly, printing to stdout and exiting. Exits with code 1 if an unknown flag is given or no filename is provided.

  • parameter argv

    The argument vector, typically Sys.argv.

  • returns

    A t with all flags and the input filename populated.
