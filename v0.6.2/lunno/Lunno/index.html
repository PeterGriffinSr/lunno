val parse_string : string -> Lunno_frontend.Ast.program

parse_string s parses the input string s and returns the resulting AST program.

  • parameter s

    The input source code as a string.

  • returns

    The AST program parsed from the input string.

  • raises Error.LexerError

    Raised if a lexical error is encountered during tokenization.

  • raises Error.ParserError

    Raised if a syntax error is encountered during parsing.

val parse_file : string -> string array * Lunno_frontend.Ast.program

parse_file filename reads the contents of the file filename, parses it, and returns the resulting AST program.

  • parameter filename

    The path to the source code file.

  • returns

    The AST program parsed from the file.

  • raises Error.LexerError

    Raised if a lexical error is encountered during tokenization.

  • raises Error.ParserError

    Raised if a syntax error is encountered during parsing.

val typecheck_program : string array -> Lunno_frontend.Ast.program -> unit

typecheck_program lines program typechecks program, printing a type error and exiting if one is found.

  • parameter lines

    The source lines, used for error reporting.

  • parameter program

    The AST program to typecheck.

  • raises Error.TypeError

    Raised if a type error is encountered.
